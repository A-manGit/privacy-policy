Privacy Policy

Effective Date: 30/04/2026

This Privacy Policy describes how this Chrome extension RecordDA handles user data.

1. Overview

The Extension is designed to capture and analyze network activity (such as HTTP/HTTPS requests, HAR logs, and media resources) from the active browser tab for debugging and testing purposes.

2. Data Collection

The Extension does not collect, store, or transmit any personal data to external servers.

All data processed by the Extension, including:

  • Network requests and responses
  • HAR (HTTP Archive) data
  • Media and video resources
  • Console logs

is handled locally within the user’s browser.

3. How Data is Used

Data is used solely for the following purposes:

  • Displaying network activity to the user
  • Generating HAR files and logs
  • Allowing users to download captured data (e.g., logs, media files)

All actions are performed only when explicitly initiated by the user (e.g., clicking “Start Recording”).

4. Permissions Usage

  • Debugger Permission: Used to capture network and console events from the active tab.
  • Tabs Permission: Used to identify and access the active tab.
  • Downloads Permission: Used to download generated files.
  • Storage Permission: Used to store logs and preferences locally.
  • Offscreen Permission: Used for background processing like ZIP generation.
  • Alarms Permission: Used for background cleanup tasks.
  • Host Permissions: Used to capture network activity from user-selected websites.

5. Data Sharing

The Extension does not share any data with third parties.
No data is transmitted outside the user’s device.

6. User Control

  • Data collection starts only when the user initiates it
  • Users can stop recording at any time
  • Downloaded data is stored only on the user’s device

7. Data Security

All processing occurs locally within the browser environment. No external servers are involved, reducing the risk of unauthorized access.

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page.
